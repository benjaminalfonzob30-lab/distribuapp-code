🛒 DistribuApp — Aplicación Móvil de Distribución de Alimentos
📱 Descripción del Proyecto

DistribuApp es una aplicación móvil desarrollada en Android que permite gestionar pedidos de una empresa de distribución de alimentos.

Sus principales funcionalidades son:

Registro e inicio de sesión mediante correo electrónico o cuenta Google
Visualización y selección de productos
Cálculo automático del costo de despacho según distancia y monto de compra
Monitoreo de temperatura del camión de reparto para asegurar la cadena de frío
📦 Reglas de Negocio
Compras iguales o superiores a $50.000 con despacho dentro de 20 km → envío gratuito
Compras entre $25.000 y $49.999 → $150 por kilómetro recorrido
Compras inferiores a $25.000 → $300 por kilómetro recorrido
🛠️ Tecnologías Utilizadas
Android Studio (entorno de desarrollo)
Java (lenguaje de programación)
Firebase Authentication (autenticación de usuarios)
Firebase Realtime Database (almacenamiento de datos)
Google Play Services (geolocalización GPS)
📲 Requisitos del Sistema
Dispositivo Android (mínimo versión 5.0)
Conexión a internet
Permisos de ubicación (GPS)
⚙️ Funcionamiento General

La aplicación permite a los usuarios registrarse, iniciar sesión y realizar pedidos.
Además, utiliza la ubicación GPS del usuario para calcular automáticamente el costo de despacho.

Por otra parte, el sistema permite monitorear la temperatura del transporte, lo que es clave para productos que requieren cadena de frío, como carnes.

📝 Historias de Usuario (Ejemplos)
Como usuario, quiero iniciar sesión para acceder a la aplicación
Como cliente, quiero calcular el costo de despacho para saber cuánto pagar
Como administrador, quiero monitorear la temperatura para asegurar la calidad de los productos
